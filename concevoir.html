Concevoir la partie GEII d’un système

La compétence “Concevoir la partie GEII d’un système” consiste à :

  • Analyser le cahier des charges fonctionnel
  • Réaliser les schémas électroniques (schéma électrique, schéma de câblage)
  • Choisir les composants (microcontrôleurs, capteurs, interfaces, alimentation)
  • Modéliser l’architecture matérielle et logicielle (diagrammes UML, bloc diagram)
  • Effectuer les simulations (SPICE, simulateur de microcontrôleur)
  • Rédiger la documentation technique associée (note de calcul, dossier de conception)

Exemples de réalisations :

  • Conception d’un capteur de température avec interface I²C pour réseau industriel
  • Définition d’une carte électronique pour la gestion d’énergie solaire (MPPT, onduleur)
  • Création d’un prototype embarqué de système d’acquisition de données
